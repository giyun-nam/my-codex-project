# Swim Pool (Prototype)

서울 구로구에 위치한 주요 수영장을 네이버 지도 위에 표시해 주는 안드로이드 앱 프로토타입입니다. 사용자는 네이버 지도에서 각 수영장의 위치를 확인하고, 마커를 눌러 주소·설명·문의처 등의 정보를 살펴볼 수 있습니다.

## 주요 기능
- **네이버 지도 연동**: 네이버 지도 SDK를 이용해 지도 화면을 표시하고, 초기 카메라를 구로구 중심부로 이동합니다.
- **수영장 위치 마커**: 구로구 내 수영장 정보를 기반으로 마커를 띄우고, 클릭 시 하단 카드로 상세 정보를 보여줍니다.
- **카메라 초기화 버튼**: 상단의 버튼으로 카메라를 다시 구로구 전체가 보이는 위치로 이동할 수 있습니다.

## 개발 환경
- Android Studio Jellyfish 이상 권장
- Android Gradle Plugin 8.7
- Kotlin 2.0
- minSdk 26, targetSdk 34

## 네이버 지도 API 키 설정 방법
1. [네이버 클라우드 플랫폼](https://www.ncloud.com/)에서 Maps > Static Map & Dynamic Map 서비스를 활성화하고 클라이언트 아이디(Client ID)를 발급받습니다.
2. 프로젝트 루트에 있는 `local.properties` 파일(존재하지 않으면 생성)을 열고 아래 항목을 추가합니다.
   ```properties
   NAVER_MAP_CLIENT_ID=발급받은_클라이언트_ID
   ```
   이 값은 Gradle 빌드 시 자동으로 `resValue`로 주입되어 `R.string.naver_map_client_id`를 통해 앱에서 접근합니다.
3. API 호출 한도 등을 확인해 필요한 경우 결제/한도 설정을 완료합니다.

## 빌드 & 실행
1. Android Studio로 프로젝트를 열고 Gradle 싱크를 실행합니다.
2. `app` 모듈을 실행 대상 기기에 설치합니다.
3. 앱이 실행되면 지도가 로드되고 구로구 수영장 마커가 표시됩니다.
   - 마커를 선택하면 하단 카드에서 수영장 상세 정보를 확인할 수 있습니다.
   - 상단 버튼으로 지도를 초기 위치로 되돌릴 수 있습니다.

## 테스트 및 향후 과제
- CI 환경에서 Android SDK가 설치되어 있지 않아 Gradle 래퍼 생성과 빌드를 수행하지 못했습니다. 로컬 개발 시 Android Studio가 자동으로 필요한 SDK와 래퍼 파일을 구성합니다.
- 프로토타입 완성 후에는
  - 위치 정보 검증 및 데이터 갱신 자동화,
  - 사용자 위치 권한 요청 및 길찾기 연결,
  - 구/군 필터링 UI 추가 등을 확장 기능으로 고려할 수 있습니다.
